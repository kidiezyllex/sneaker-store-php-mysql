- Các chức năng chính:
### Customer
- Đăng ký / Đăng nhập / Đăng xuất
- Xem danh sách sản phẩm (lọc, tìm kiếm, phân trang)
- Xem chi tiết sản phẩm (hình ảnh, mô tả, size, giá)
- Thêm / Xóa / Cập nhật giỏ hàng
- Thanh toán đơn hàng (chọn phương thức, nhập địa chỉ)
- Theo dõi trạng thái đơn hàng
- Đánh giá / Nhận xét sản phẩm
- Quản lý thông tin cá nhân (profile, đổi mật khẩu)

### Admin
- Đăng nhập / Đăng xuất
- Quản lý sản phẩm (thêm, sửa, xóa, cập nhật kho)
- Quản lý danh mục / thương hiệu / size
- Xem và xử lý đơn hàng (xác nhận, hủy, cập nhật trạng thái)
- Quản lý khách hàng (xem, khóa tài khoản)
- Xem báo cáo doanh thu / thống kê đơn hàng
